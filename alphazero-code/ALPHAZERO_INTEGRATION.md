# AlphaZero Integration - Готово! ✅

## Что добавлено:

1. ✅ **Новый уровень сложности "AlphaZero"** в интерфейсе
2. ✅ **ONNX модель** экспортирована (89 KB)
3. ✅ **Интеграция с браузером** через ONNX Runtime Web
4. ✅ **MCTS с нейросетью** для сильных ходов

## Как использовать:

1. Открой `index.html` в браузере
2. Выбери режим "vs Бот"
3. Выбери уровень **"AlphaZero"** (последняя кнопка)
4. Нажми "Ойынды бастау"

## Технические детали:

- **Модель**: обучена на 5000 игр, 50 итераций
- **Win rate**: 90-100% против случайного игрока
- **MCTS симуляций**: 400 на ход
- **Время на ход**: ~2-5 секунд (зависит от браузера)

## Файлы:

- `model.onnx` - обученная нейросеть (89 KB)
- `metadata.json` - метаданные модели
- `alphazero-inference.js` - JavaScript обёртка для ONNX

## Примечания:

- Первая загрузка модели может занять 1-2 секунды
- Модель кэшируется браузером после первой загрузки
- Для максимальной скорости используй Chrome/Edge (лучшая поддержка WebAssembly)

## Оптимизации (опционально):

Если хочешь ускорить:
- Уменьши `simulations` в `AI_LEVELS.alphazero` (например, до 200)
- Или увеличь для большей силы (до 800)
