## Проблемы

1. **Недостаточно симуляций MCTS**
   - AlphaZero использует только 200 симуляций
   - Противники используют 5000-100000 симуляций
   - Нейросеть не может компенсировать такой разрыв

2. **Модель недостаточно обучена**
   - Всего 50 итераций, 5000 игр
   - Для сильной игры нужно 100+ итераций, 10000+ игр

3. **Несоответствие симуляций**
   - При обучении использовалось 200 симуляций
   - При тестировании против сильных оппонентов нужно больше

## Текущий уровень AlphaZero

**Оценка:** Слабее Hard (Minimax 6), примерно на уровне Easy-Medium

**Причины:**
- Модель обучена только на 50 итерациях
- Использует мало симуляций (200)
- Нейросеть ещё не научилась достаточно хорошо оценивать позиции
